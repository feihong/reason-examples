# Feihong's Reason Examples

## Prerequisites

    yarn global add nodemon npm-run-all

## Commands

Install dependencies

    yarn

Build

    yarn build

Watch

    yarn watch

Build in VS Code

    Windows + Shift + B

List all programs

    yarn go

Build and run a program

    yarn go demo

Automatically build and run a server program

    yarn dev server
